# Anime News Bot

Telegram-бот для автопостинга аниме-новостей в канал/ветку обсуждения.

## Возможности

- Сбор новостей с 14 источников (RSS + HTML парсинг)
- Перевод на русский с защитой названий (Google Translate + AniList API)
- Дедупликация, фильтры по свежести и наличию картинок
- Blacklist стоп-слов (реклама/товарка)
- Режим отправки в ветку обсуждения (всё пачкой) или в канал (по одному)
- Догрузка полного текста статей со страниц
- Метрики, логирование, retry-логика

## Запуск

### Переменные окружения

Обязательная:
- `BOT_TOKEN` — токен бота от @BotFather

Опциональные (есть значения по умолчанию):
- `CHANNEL_ID` — ID/username канала
- `ADMIN_ID` — Telegram ID администратора
- `DISCUSSION_CHAT_ID` — ID группы обсуждения
- `DISCUSSION_THREAD_ID` — ID темы (ветки)
- `DATA_DIR` — папка для хранения данных (для постоянного хранилища на хостинге)

### Установка и запуск

```
pip install -r requirements.txt
python anime_news_bot.py
```

## Тесты

```
pip install -r requirements-dev.txt
pytest
```

## Команды бота

- `/start` — меню
- `/news` — свежие новости в личку
- `/preview` — превью постов
- `/start_auto`, `/stop_auto` — управление авторассылкой
- `/status` — статус
- `/stats` — метрики
- `/logs` — последние строки лога
- `/blacklist` — список стоп-слов
- `/chatinfo` — ID чата и темы
- `/settings` — настройки
